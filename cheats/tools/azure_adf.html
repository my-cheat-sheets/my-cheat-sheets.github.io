Azure Data Factory (ADF) Cheatsheet

Basics

Core Concepts

      Pipeline

      Logical grouping of activities that perform a unit of work.

      Activity

      Processing step in a pipeline (e.g., Copy Data, Notebook, Stored Procedure).

      Dataset

      Represents data structures within data stores (table, file, folder).

      Linked Service

      Connection string/information to external resources (SQL DB, Blob Storage).

      Integration Runtime (IR)

      Compute infrastructure used by ADF (Azure, Self-Hosted, SSIS-Azure).

Common Activities

      Copy Data 
      
Source: Blob, SQL, API, etc.
Sink: Blob, SQL, Lake, etc.
Mapping: Explicit or Implicit
      Control Flow
      • Execute Pipeline: Invoke another pipeline.
      • ForEach: Loop over a collection.
      • If Condition: Branching logic.
      • Wait: Pause execution.
      • Web: Call REST API.

      Functions 
      
@concat('a', 'b')
@formatDateTime(utcnow(), 'yyyy-MM-dd')
@equals(variables('status'), 'Succeeded')
@coalesce(activity('Act1').output.value, 'default')
      Activity Output 
      
@activity('LookupActivity').output.count
@activity('LookupActivity').output.value[0].ColumnName

Triggers

      Schedule Trigger

      Wall-clock schedule (e.g., every day at 8 AM).

      Tumbling Window

      Operates on time slices, supports backfill and dependency.

      Storage Event

      Triggered by Blob Created/Deleted events (Event Grid).
